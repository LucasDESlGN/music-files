# Music Files

This repository is used to store MP3 audio files for my personal projects.

The files are hosted on GitHub and accessed through direct raw links, making them easy to use in applications, web projects, and game modifications.

## Features

- MP3 audio hosting
- Stable direct links
- Simple file organization
- Easy file management

## File Format

```
01.mp3
02.mp3
03.mp3
04.mp3
```

Each file can be accessed using the following pattern:

```
https://raw.githubusercontent.com/LucasDESlGN/music-files/main/01.mp3
```

Replace the file number to access a different audio.
